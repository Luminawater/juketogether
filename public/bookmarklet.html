Unfortunately, localtunnel's password page can't be fully bypassed because it checks your browser's User-Agent before any JavaScript runs.

Solution 1: Browser Extension (Easiest)

Install a browser extension to modify headers:

  • Chrome/Edge: Install "ModHeader" extension
  • Firefox: Install "Modify Headers" extension

Then add this header: bypass-tunnel-reminder: true

Solution 2: Use ngrok Instead

ngrok doesn't require passwords (but needs a free account):

  1. Sign up at https://ngrok.com
  2. Get your authtoken from the dashboard
  3. Run: ngrok config add-authtoken YOUR_TOKEN
  4. Run: npx --yes ngrok http 8080

Solution 3: Accept the Password

The password is only needed once per visitor's IP address every 7 days.
